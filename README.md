# tutorial-examples

## Verifier integration demo

1. JS example
    - `cd verifier-integration/js`
    - `npm install`
    - `node index.js`
2. Go example
    - `cd verifier-integration/go`
    - `go run index.go`

For additional info please visit [Polygon ID tutorial](https://devs.polygonid.com/docs/quick-start-demo)
